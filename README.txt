mvn compile - kompilerar projektets källkod och skapar klassfiler i mappen target/classes

mvn clean - tar bort alla skapade filer från tidigare builds

mvn javadoc:javadoc - skapar Javadoc-dokumentation

mvn surefire-report:report - skapar en rapport baserad på resultat från tester

mvn test - kör projektets testklasser

mvn package - paketerar projektets kompilerade källkod, resurser och beroenden till en jar-fil
